# Defolder Developer Kit

Пока что Editor script для Defold, позволяющий удалять комментарии и строки, начинающие на `print()` из `Lua/(render, gui, editor, defold)script` файлов.

## Установка

Добавь в `game.project` в секцию `dependencies`: [[ссылка на релизную версию]](https://github.com/dprogrb/ddk/releases/new)
